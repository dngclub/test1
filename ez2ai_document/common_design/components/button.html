ez-tailwind.css 정의 - 모든 페이지가 사용

표준 변형 (variant)

<button className="eg-btn eg-btn-primary">저장</button>
<button className="eg-btn eg-btn-default">취소</button>
<button className="eg-btn eg-btn-outline">+ 등록</button>
<button className="eg-btn eg-btn-danger">삭제</button>

크기

표준은 헤더/검색/모달 푸터, 작은 크기는 그리드 셀 내부 (행 액션 등) 에만.

아이콘 버튼

상태

전체 너비 (모달 푸터 등)

<button className="eg-btn eg-btn-block eg-btn-primary">로그인</button>

사용 가이드

  • primary = "이 화면의 주 액션" (저장, 로그인, 검색) - 한 영역에 1 개만
  • default = 보조 액션 (취소, 닫기, 부 기능)
  • outline = "추가/생성" 의미가 강한 보조 (+ 등록)
  • danger = 삭제, 거부 - 모달 footer 좌측 (className="eg-btn eg-btn-danger left")
  • success / warning = 결재 승인/보류, 상태 변경
  • ghost / link = 그리드 셀 내부 인라인 액션 등 최소 영역
