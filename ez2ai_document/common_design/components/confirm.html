삭제 확인 (가장 자주 사용)

삭제 확인

"공조기 #02 정기점검" 오더를 삭제하시겠습니까? 이 작업은 되돌릴 수 없습니다.

import { ConfirmModal } from '../M000000000/shared'

const [confirm, setConfirm] = useState<{title:string; msg:string; on:()=>void}|null>(null)

// 삭제 버튼 클릭
setConfirm({
  title: '삭제 확인',
  msg: `"${row.title}" 오더를 삭제하시겠습니까?\n이 작업은 되돌릴 수 없습니다.`,
  on: () => deleteRow(row.id),
})

// JSX
<ConfirmModal
  open={!!confirm}
  title={confirm?.title}
  message={confirm?.msg}
  onConfirm={() => { confirm?.on(); setConfirm(null) }}
  onCancel={() => setConfirm(null)}
  confirmLabel="삭제"
  confirmDanger
/>

경고/주의

⚠️ 미저장 변경사항

저장하지 않은 변경사항이 있습니다. 페이지를 이동하면 변경 내용이 사라집니다.

가이드

  • 삭제 → confirmLabel="삭제" confirmDanger (danger 버튼)
  • 저장/승인 → confirmLabel="저장" (primary 버튼)
  • 본문에 "되돌릴 수 없습니다" 명시 (삭제 시)
  • 대량 처리는 건수까지 표시 ("12 건을 삭제하시겠습니까?")
