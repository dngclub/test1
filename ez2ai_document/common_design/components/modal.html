가이드

  • 제목: 신규는 "○○○ 등록", 수정은 "○○○ 수정 - {id}"

  • view 모드에서는 input 대신 <div className="eg-form-view">...</div>
  • 모달 크기: 기본 720, 짧은 폼 size-sm(480), 다중 컬럼 size-lg(960)
