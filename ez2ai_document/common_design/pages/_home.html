이 디자인 시스템에 대하여

ez2AI 공통 시스템 의 모든 화면이 따라야 할 색·폰트·간격·컴포넌트 기준을 한 곳에 모은 정적 HTML 쇼케이스입니다. 실제 React 화면 (web/src/M000000XX/) 과 본 쇼케이스는 동일한 디자인 토큰 (--ez-*) 과 동일한 컴포넌트 클래스 (eg-*, xx-*) 를 사용합니다.

📌 신규 메뉴를 개발할 때 반드시 본 페이지의 컴포넌트와 샘플을 먼저 클릭해 본 뒤, 해당 클래스/구조를 그대로 따라 React 컴포넌트에 적용하세요. AntD/MUI/shadcn 등 외부 UI 라이브러리 도입 금지.

Brand Navy #003087 1색 중심. 상태색은 토큰 4종(success/warning/danger/info)만.
가독성
본문 14px, 라벨 13px, 컨트롤 36px - 레거시보다 + 2~4px 키워 노안 대응.

헤더 sticky + tint 배경 + tabular numbers. 행 클릭 → 수정 모달 일관.
